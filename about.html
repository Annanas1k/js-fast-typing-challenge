Test de Tastare

Această aplicație web este un test de tastare care ajută utilizatorii să își măsoare viteza de tastare în cuvinte pe minut (WPM) și precizia. Aplicația furnizează un text aleatoriu de tastat, iar utilizatorii trebuie să tasteze textul corect într-un interval de timp dat (1 minut). Testul urmărește și afișează erorile și viteza de tastare în timp real, iar la final, utilizatorul poate vedea statistici despre performanța sa.

Funcționalități principale:

  • Test de Tastare: Permite utilizatorilor să efectueze un test de tastare cu un timp de 1 minut.
  • Calcularea Vitezei: Viteza de tastare este calculată în cuvinte pe minut (WPM).
  • Monitorizarea Erorilor: Se afișează numărul de erori în timp real.
  • Rezultate și Istoric: Se salvează viteza, precizia și erorile într-un istoric.
  • Fapte Inutile: Se afișează un fapt inutil aleatoriu între teste.

Tehnologii utilizate:

  • HTML: Structura paginii.
  • CSS: Stilizarea paginii.
  • JavaScript: Logica aplicației, calcul WPM, erori, etc.
  • Local Storage: Salvarea rezultatelor testelor.

Cum să utilizezi aplicația:

  1. Clonare repo:
    git clone <repository-url>
  2. Deschide aplicația:
    Deschide fișierul index.html în browserul tău preferat.
  3. Fă un test:
    Apasă butonul „Start”, tastează cât mai rapid și precis.
  4. Vezi istoricul testelor:
    Poți accesa o listă cu testele tale anterioare.
  5. Șterge istoricul:
    Folosește butonul de ștergere dacă vrei să golești istoricul.

Descriere detaliată a funcționalităților:

Test de Tastare:

Aplicația selectează un text aleatoriu din texts.json și îl afișează. Utilizatorul trebuie să tasteze textul într-un interval de 60 de secunde. Viteza (WPM) și erorile sunt afișate în timp real.

Calculul Vitezei și Preciziei:

La finalul testului, se calculează WPM (cuvinte pe minut) și precizia (caractere corecte din total).

Istoricul Testelor:

Rezultatele fiecărui test sunt salvate în localStorage și pot fi vizualizate ulterior împreună cu data, viteza și erorile.
